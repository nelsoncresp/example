# Contexto de Diseño: Consultorio Psicológico SPA

Este documento detalla la identidad visual, los principios de diseño y la estructura técnica del proyecto de modernización de la plataforma web del consultorio psicológico.

## 1. Visión General
El objetivo es transformar una SPA (Single Page Application) funcional en una experiencia digital **Premium**, que transmita autoridad, confianza y empatía. Se ha priorizado un diseño personalizado que se aleje de los patrones genéricos generados por IA.

## 2. Identidad Visual (Brand Identity)

### Paleta de Colores
- **Púrpura Principal:** `#26104f` (Navbar y elementos de autoridad).
- **Púrpura de Acento:** `#45207f` (Iconografía y detalles).
- **Fondo:** Blanco puro (`#ffffff`) para máxima limpieza visual.
- **Texto:** Escala de morados profundos para evitar el negro genérico y mantener la coherencia de marca.

### Tipografía
- **Fuente Principal:** *Outfit* (Google Fonts).
- **Estilo:** Pesos pesados (Bold/ExtraBold) para títulos y pesos medios (Medium) para cuerpo, buscando un look editorial y moderno.

## 3. Principios de Diseño "Anti-IA"

El proyecto rechaza los vicios comunes del diseño web contemporáneo automatizado:
1. **Flat Design vs. Sombras:** Se han eliminado las sombras suaves y degradados genéricos en favor de un diseño plano y sólido que se apoya en la tipografía y el color puro.
2. **Monocromía con Intención:** El uso del morado no es solo decorativo, sino que reemplaza al gris y negro tradicional para crear una atmósfera de marca única.
3. **Minimalismo de Impacto:** Menos elementos visuales, pero con mayor peso y propósito (bordes finos, espaciado generoso).

## 4. Estructura de Secciones Clave

### Matriz DOFA (Análisis Estratégico)
- **Diseño:** Tarjetas minimalistas con cabeceras en tonos pasteles muy sutiles.
- **Innovación:** Uso de tipografía monocromática morada en todo el contenido para un acabado de lujo.
- **Simetría:** Cuadrícula de 2x2 perfectamente equilibrada en escritorio.

### Servicios Terapéuticos y Equipo Profesional
- **Layout:** Reconfiguración a un sistema de cuadrícula **2x2 (col-lg-6)** para garantizar que todas las tarjetas tengan el mismo peso visual y evitar elementos huérfanos.
- **Consistencia:** Alineación perfecta de imágenes y textos para un flujo de lectura profesional.

## 5. Especificaciones Técnicas
- **Estructura:** HTML5 Semántico.
- **Estilos:** CSS3 nativo (Vanilla CSS) con variables para una gestión ágil de la identidad.
- **Grid:** Bootstrap 5 (utilizado estrictamente para el sistema de columnas y responsividad).
- **Animaciones:** `Animate.css` para revelaciones suaves (`fadeInUp`) al hacer scroll.
